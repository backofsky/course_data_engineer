## Практическая работа №14. Машинное обучение на больших данных

Необходимо выполнить следующие действия:

-	Скачайте данные по ссылке: https://www.kaggle.com/competitions/titanic/data?select=train.csv.

-	Исключите признаки, которые на ваш взгляд, могут привести к переобучению (например, id).

-	Разделите выборку на train и test.

-	Преобразуйте категориальные признаки с помощью sklearn.preprocessing.OneHotEncoder, а некатегориальные с помощью sklearn.preprocessing.StandardScaler (на следующем занятии расскажем, зачем это нужно)

-	Обучите модели логистической регрессии и KNN, посчитайте все метрики классификации, которые были упомянуты на занятии, сделайте выводы.

-	Подберите гиперпараметры для обоих моделей с помощью RandomizedSearchCV, напишите, почему, как вы думаете, эти параметры подошли.

-	Посчитайте метрики для новых моделей, также сделайте выводы.
